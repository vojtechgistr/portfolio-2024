Face Recognition SystemLATEST

Real-time face identification system with over ~99.36% accuracy, including 3D analysis, face adaptation and multiple-phase verification. The speed achieves an average of ~19.3 ms with 10,000 faces in a PostgreSQL database. An Intel RealSense D435 camera is integrated into the system, which is written in C# .NET 8 and was developed for a partner of SPC Solutions s.r.o.

Nanoleaf® adaptation - Modular LED lights that can be controlled using a mobile cross-platform app written in React Native. In this project I learned how to work with electronics, synchronization and communication between specific devices. The receiver is written in C++.

Battleships Minigame

Battleships is a 2-player strategy game in which you play against the AI. Each player has different types of ships placed on their field that the opponent cannot see. The goal is to shoot down all the ships. Making this game was a good way to learn how to create more complex algorithms and a highly interactive AI. The console application was written in C++.

A chatbot that has been used by 147 Discord servers and 180,000+ users in its lifetime. It featured sophisticated commands for moderation, entertainment, chatting, and high-quality music playback in voice channels. It was written in Node.js and communicated with the Discord API.

Let's Study - Web App

Discord partner Let's Study needed statistics, user management, blogging, study materials and much more. All of this is now managed by one web application. In 2018, I attended a camp at Tomáš Baťa University in Zlín, where I built my first Arduino lego robot and won 3rd place in the university competition, which led me to become more interesed in creating my own applications and systems. Since then I've been doing personal and volunteer projects such as Let's Study. To this day, I am always looking for new knowledge and experiences, which I enjoy.

My expertise lies in software development and design. I have experience in Full-Stack web & app development, APIs, algorithms, databases and configuring Ubuntu systems.
